Usage

1. Load the Widget

Place this block this inside your header or after your body tag.

<script>
    (function (w,d,s,o,f,js,fjs) {
        w['YouChess-Widget']=o;w[o] = w[o] || function () { (w[o].q = w[o].q || []).push(arguments) };
        js = d.createElement(s), fjs = d.getElementsByTagName(s)[0];
        js.id = o; js.src = f; js.async = 1; fjs.parentNode.insertBefore(js, fjs);
    }(window, document, 'script', 'cw', '//chessdiagram.herokuapp.com/widget.js'));
</script>

2. Set your preferences

This block is optional. If not specified values will fallback to defaults.

<script>
    cw('init', { 
        interval: 750,
        speed: 150,
        pieces: 'uscf',
        whitesquare:'#f0d9b5',
        blacksquare:'#b58863',    
        lang: 'es',
        autoplay: true
    });
</script>
Parameter Datatype Default Description
interval Integer 1000

Time between moves. In milliseconds.
speed Integer 200

Piece animation speed. In milliseconds.
pieces String 'classic'

Pieces theme. Available pieces styles are 'classic', 'chess24', 'neo', 'neon', 'uscf', 'wood', 'book', 'leipzig', 'cases' and 'modern'.
whitesquare String '#f0d9b5'

Custom light squares background color.
blacksquare String '#b58863'

Custom dark squares background color.
lang String 'en'

Language to describe the diagram. Available languages are en and es.
autoplay Boolean false

The widget will start automatically after being loaded if set to true.

ECO

Chess Diagram will automatically detect and add the Encyclopedia Chess Opening (ECO) for your game.

Annotations

Following PGN standards annotations are grouped inside curly braces.

1.e4 e5 2.Nf3 Nc6 3.Bb5 {This will be your annotation.} Nf6 4.d3 ...
